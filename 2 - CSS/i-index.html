Selectors

Seletores do CSS são responsáveis por determinar quais elementos do HTML receberão estilização

Tipos mais comuns:

  • Por tipo: busca elementos por uma tag HTML especifica.
    • Exemplo de seletor por tipo.
  • Por ID (#): busca elementos através do atributo "id".
    • Exemplo de seletor por id.
  • Por classe (.): busca elementos através do atributo "class".
    • Exemplo de seletor por classe.
  • Universal (*): seleciona todos os elementos HTML.
  • Por atributo ([atrib]): seleciona elementos que possuem um atributo específico em nosso documento HTML ou/e buscar atributos com um valor específico.
    • Exemplo de seletor por tipo.
